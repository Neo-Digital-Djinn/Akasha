# Akasha Requests

This repository is the intake system for future Akasha capabilities.

Requests captured here may become:

- new organs
- experiments
- discovery engines
- research directions
- infrastructure components

The goal is to capture ideas **quickly and structurally** so they can be explored later.

---

## Workflow

idea appears  
↓  
capture request JSON  
↓  
commit to `requests/open/`  
↓  
review or refine  
↓  
move to exploring / approved  
↓  
Forge materializes repo

---

## Status Flow

open → exploring → approved → forged → archived

open  
Raw captured idea

exploring  
Actively discussing or refining

approved  
Ready to become a repo or system

forged  
Implemented elsewhere

archived  
Stored but inactive

---

## Philosophy

Akasha grows by **organ requests**.

Ideas are treated as seeds rather than tasks.

Not every seed must grow, but every seed should be preserved.

## Engine Role

Akasha Requests is the intake engine of the constellation. It captures structured request objects and moves them through a lifecycle toward potential realization.

## Why it exists

Ideas arrive faster than they can be implemented. This system preserves those ideas long enough to classify, evaluate, and eventually materialize them as engines, repairs, or experiments.

## Inputs

- request JSON files
- translated human proposals
- discovery engine suggestions
- repair requests
- infrastructure expansion ideas

## Memory / Registry

The request registry stores lifecycle states:

- open
- exploring
- approved
- forged
- archived

It also maintains request indexing and lineage.

## Relation Model

Akasha Requests sits upstream of Forge and downstream of translators, mutation engines, and discovery emitters.

## Evaluator

Requests are evaluated according to:

- structural completeness
- alignment with constellation goals
- clarity of intent
- readiness for forging
- approval state

## Outputs

- staged request objects
- approved forge candidates
- request index updates
- archived idea history

## Position in Constellation

This repo acts as the intake valve of the Akasha constellation.

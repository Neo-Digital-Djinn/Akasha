## Install shape
The zip roots are arranged so they can be extracted into `~/`.

- JSON zip places files into `~/akasha-requests/requests/open/`
- Heredoc wrapper zip places scripts into `~/akasha-requests/heredoc_wrappers/open/`
- Install script expects both zips to live in `/sdcard/Download/`

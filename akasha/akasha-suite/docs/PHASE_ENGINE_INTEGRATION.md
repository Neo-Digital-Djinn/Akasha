
# Akasha Phase Engine Integration

The Phase Engine is now vendored into this root as `akasha-phase-engine/`.

## Quick Start

```bash
./bin/akasha-phase
./bin/akasha-phase discovery
./bin/akasha-phase graph
```

## Why it matters

This engine models states, transitions, topology, emergence, and candidate undiscovered phases.
It can later generalize beyond physics into markets, ecosystems, civilizations, and symbolic domains.
